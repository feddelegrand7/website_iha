Dynamic Data Tables with DT

Last updated on Aug 14, 2019 2 min read 
library(widgetframe)

The DT package offers nice visualizations for data tables. In this example, we’ll use the mtcars dataset which is described [here] (https://stat.ethz.ch/R-manual/R-devel/library/datasets/html/mtcars.html).

Let us first of all load our package:

library(DT)

Then we use the datatable() function to display the mtcars in an interactive way:

table1 <- datatable(mtcars)

table1
widgetframe::frameWidget(table1)

You can see that the above table provides with a search and sort functionalities which can be convenient for a larger dataset. Moreover, one can set the number of desirable entries to display (10, 25, 50 or 100).

Furthermore, we can include a filter functionality which can be displayed either at the top or at the bottom of the table.

datatable(mtcars, filter = "bottom")
