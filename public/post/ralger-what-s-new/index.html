ralger: what's new ?

Last updated on Feb 19, 2020 1 min read

The new version of ralger is on CRAN !

So, what’s new ?

askRobot

With ralger 1.1.0, you have now the ability to consult the robots.txt of the page web you want to scrap in order to check if you have the right to do so or not. This wont stop the extrancting process, but you’ll receive a message indicating whether you’re eligible or not to do so. To illustrate this fact, let’s dive into an example and extract a very interesting information: the 25 most powerful one piece characters !

library(ralger)

link <- "https://www.cbr.com/one-piece-characters-ranked/"

scrap(link, node = ".item-title", askRobot = T)
